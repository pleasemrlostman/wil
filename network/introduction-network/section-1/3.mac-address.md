# MAC 주소

### 한줄요약 : 하드웨어에 할당된 주소

## Mac Address 란?

- Media Access Control의 약자로서 **네트워크 통신을 하는 하드웨어에 할당된 주소**
- IP주소가 네트워크 주소라면 맥 주소는 하드웨어 주소, 물리적 주소, 이더넷 주소
- 실제로는 중복된 맥 주소도 존재하고, 맥 주소를 변경 가능한 네트워크 장비도 존재
    - 그러나 맥 주소를 사용해서 통신을 하는 동일 링크에서는 고의적으로 설정을 변경하지 않는 이상 맥 주소가 충돌하는 경우는 존재하지 않음
- 맥 주소는 12자리 문자열, 48비트, 16진수로 표기
    - `02:46:8A:CE:00:FF` 또는 `02-46-8A-CE-00-FF`로 표기
    - 처음 6자리는 어댑터 제조 업체, 마지막 6자리는 해당 특정 어댑터의 고유 식별 번호
- 맥 주소는 왜 필요한가?
    - 네트워크 통신은 IP주소만 가지고 통신을 하는 것처럼 보이지만 실제로는 마지막에 맥 주소가 필요하다
        - TCP/IP 또는 OSI 7 Layer라는 통신 표준에 설계되어 있는대로 통신하기 때문
        - 아이피 주소는 레이어3 네트워크에서 사용, 맥 주소는 레이어 2 데이터 링크에서 사용
        - 통신 표준에 맞게 단계별로 통신이 진행 될 때, 아이피주소에서 맥 주소로 변환되는 과정을 거침 ⇒ **ARP(Address Resolution Protocol)**

## Mac 주소를 확인 하는 방법

1. cmd 창 접속 (또는  `getmac -v` 입력)
2. ipconfig/all 명령어 입력
